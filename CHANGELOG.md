- Responsive design (mobile-first)
- OpenGraph image generation

- Next.js 16.x (App Router, SSG)
- React 19.x
- TypeScript 5.x
- Tailwind CSS 4.x
- Framer Motion 12.x
- Shadcn/ui (New York style)
- Lenis 1.3.x (smooth scrolling)
